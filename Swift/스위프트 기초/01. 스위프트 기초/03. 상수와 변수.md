# 1. 상수와 변수 선언

- **let**: 상수 선언 키워드
- **var**: 변수 선언 키워드

```Swift
// 상수의 선언
let 이름: 타입 = 값

// 변수의 선언
var 이름: 타입 = 값

// 값의 타입이 명확하다면 타입 생략 가능
// 그래도 타입 명확히 적어주는 것 권장
let 이름 = 값
var 이름 = 값

// 변수는 추후에 다른 값을 할당할 수 있지만 상수는 값을 변경할 수 없다.

// 나중에 할당하려고 하는 상수나 변수는 타입을 꼭 명시해주어야 한다.
let sum: Int
let inputA: Int = 100
let inputB: Int = 200

// 상수는 선언 후 최초 1회만 할당 가능하다.
sum = inputA + inputB
```
